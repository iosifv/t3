# t3
Terminal Toolbelt Two

## Installation

##### Install Gum
`https://github.com/charmbracelet/gum`

##### Clone this repo
```bash
cd ~ && mkdir www && mkdir www/github && cd www/github/
git clone git@github.com:iosifv/t3.git
```

##### Use install script

```bash
sh installer.sh
```
